Lesson Two

Creating a Structured Document

HTML files are the structure of web pages, while CSS are what dictates what the pages look like.
Javascript are the components that change the elements of the HTML. It is important to structure a web page because it becomes much easier to read and to alter the code later. When making your own web page, it's a good idea to look at other people's code in order to better understand how to write your own code.

Tree-Like Structures

The HTML DOM model is constructed as a tree of objects. It turns tags and content into elements in the tree. All of the elements in a web page are boxes. When designing a web page from a template, the easiest way to start is by dividing all of the items into boxes. It is important to start with the biggest boxes and then proceed to the smaller ones.

Lesson Three

Adding CSS Style and HTML Structure

Cascading Style sheets define the formatting (style) to be applied to the HTML. These properties can be applied to both individual elements as well as to its descendants. CSS can be called from the HTML page itself or from its own seperate file. It is ideal to write CSS code on a separate file.

ID's, Classes and Semantic tags

ID's and classes mark elements in order to change their characteristics in CSS. While classes refer to multiple elements, ID's refer to unique elements. Headings like h1 and h2 also help avoid to repetition by allowing you to change multiple elemets using CSS. I used CSS on this webpage. Since the white rectangles that surround this text all needed to be the same, I assigned them a class element. Then in the CSS file I only needed to apply the style to the class name instead of changing every rectangle one by one inside the HTML file.

The Box Model

HTML elements are boxes and each box has 4 components; content, padding, border and margin. It is difficult to fit all of these boxes together on a web page. There are two ways for dealing with this issue. The first is to set the sizes in terms of a percentage. The second option is to set the box-sizing attribute to border-box for every element. In order to have two boxes next to each other, it is necessary to add display:flex; to your CSS code.
